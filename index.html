Game Theory Simulation

Instructions:

Play 200 rounds against a computer opponent. Choose to Cooperate or Defect each round. Try to maximize your score.

Scoring:

  • Both Cooperate: 3 points each
  • You Cooperate, Computer Defects: You 0, Computer 5
  • You Defect, Computer Cooperates: You 5, Computer 0
  • Both Defect: 1 point each
